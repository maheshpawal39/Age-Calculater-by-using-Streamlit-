# 🎂 Advanced Age Calculator

A simple and interactive **Age Calculator** built with **Streamlit** that calculates your exact age and provides additional insights such as zodiac sign, total days lived, birthday weekday, leap year status, and countdown to your next birthday.

---

## 🚀 Features

* 📅 Select your Date of Birth
* 🎂 Calculate exact age (Years, Months, Days)
* 📆 Total Days Lived
* ⏰ Approximate Hours Lived
* 🎈 Birthday Weekday
* ♈ Zodiac Sign
* 🌍 Checks if Birth Year was a Leap Year
* 🎉 Countdown to Next Birthday
* 🎈 Fun Balloon Animation after calculation
* 📱 Clean and Responsive UI

---

## 📦 Installation

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/Advanced-Age-Calculator.git
```

### 2. Navigate to the Project Folder

```bash
cd Advanced-Age-Calculator
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the Application

```bash
streamlit run app.py
```
